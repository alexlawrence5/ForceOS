ForceOS - Remastered Windows

ForceOS is an open-source Windows fork designed for performance, privacy, and open collaboration.

Community-Driven by Design

ForceOS isn't controlled by a corporation. It evolves through a community of contributors, free from proprietary restrictions.

Do ForceOS collect user data?

No, ForceOS ITSELF does NOT collect any user data. But who knows? ForceOS is based off Windows, so if Windows does...

Which people can use ForceOS?

ForceOS is for everyone. But if you are a gamer, tech-enthusiast, or just someone who values open-source software, you'll find ForceOS to be a great fit.
