# What is ForceOS?
ForceOS is an Windows fork that is debloated, optimized.

# But why?
Since Microsoft is making Windows more limited, bloated. So I decided this should end in one ISO.

# How does the source code work?
- bin/ -> Contains core app, NTLite. NTLite is an tool to debloat ISO's
- iso/ -> Contains Windows 10 21H1 ISO downloader. We didn't put directly because GitHub doesn't allow.
- helper/ -> Contains helper scripts, like GPT converter...
