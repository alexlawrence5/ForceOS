# Here is a quick guide of setupping ForceOS
NOTE: THIS PROCESS WILL DELETE ALL YOUR DATA!
- Get an empty USB stick
- Install Rufus
- Download the ForceOS ISO
- In Rufus, select the USB as your target disk (Data loss if its full!)
- In Rufus, select the ISO as ForceOS ISO
- This may take a while, so relax until the writing process is finished
- After the process is finished, open your boot manager
- Boot it from the bootable USB
- Setup the OS
- Enjoy as much as you can
